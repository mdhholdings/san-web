State Template v5.5

The San Andreas State Template is an HTML template and website standard created by the California Department of Technology to state agencies and departments within the State of California and beyond. Please visit webstandards.ca.gov for more information.

This is a modified version of the template used to create internal websites for the State of San Andreas' Agencies within GoldFieldRP. We encourage you to utilize GitHub to provide feedback and collaborate on the development of the next San Andreas State Template! For questions please email hmayotte@mdhholdings.org.

This is a living document and changes to this template, the sample and instructional pages are made daily and can be tracked on the GitHub Repository (TBA).

Note: You will be able to download each version using the GitHub repository links. Look for the "Clone or Download" button on the respective GitHub page.
