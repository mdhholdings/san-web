Update your State Template to most current version

If your website is using a State Template that is perhaps couple of revisions back, you may be able to use the following method to update. Make extra backups if you plan to use the methods outlined below.

Before you start

Make a backup of your current site before you start. This way if you make any errors you still have your current site. Most template updates are made to make State Template more ADA compliant and for better compatibility in mobile devices.

Updating steps

  1. Download the latest production or beta State Template release from our GitHub repository (TBA)
  2. Copy over the core css and colorscheme files from /css folder or compare the old and new CSS files. Most updates include changes to the CSS code.

Use State Template CDN

Update your website faster by using State Template CDN. You will only need to update CSS and JavaScript CDN links. No manual copying over files is needed. Instead, you can replace copying over files described in steps 2 and 3 with CDN. By doing so you will eliminate step 5 because fonts are already referenced in CDN import calls.
